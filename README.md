# 🛡️ Safe Markets

Safe Markets is a trust-based e-commerce platform designed to connect buyers and sellers through a secure, transparent, and reputation-driven system.

Unlike traditional marketplaces, Safe Markets integrates a **Trust Engine** that evaluates sellers based on reviews, reports, and verification status.

---

## 🚀 Project Overview

Safe Markets enables:

- Buyers to browse and purchase products confidently
- Sellers to build trust and reputation
- Admins to monitor and control platform activity

The system focuses on **fraud prevention, transparency, and accountability**.

---

## 🧠 Core Features

### 🔐 Authentication
- Email & password login
- Role selection (Buyer / Seller)

### 🛒 Marketplace
- Browse products
- View price, description
- Add to cart / Buy now
- Chat with seller

### 💬 Chat System
- Internal messaging between buyer & seller
- No external communication (keeps trust within platform)

### 🧑‍💼 Seller Verification
- Business name
- Category selection
- Phone number
- ID upload
- Approval required before selling

### ⭐ Reviews System
- Buyers leave reviews after interaction
- Reviews linked to seller (not just product)

### 🔍 Business Trust Lookup
- Search any business
- View trust score
- See all reviews

### ⚠️ Reports & Disputes
- Buyers can report sellers/products
- Reports affect trust score

### 🛠️ Admin Panel
- Approve or block sellers
- Monitor users and products
- Resolve disputes
- Enforce trust rules
